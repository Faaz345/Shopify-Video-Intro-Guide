Invalid Access Token

The access token you're using is invalid or has been tampered with.

Possible reasons:
  • The link has been modified or corrupted
  • You're using an old or expired link
  • The token was not properly generated
  • Security validation failed

How to fix this:

  • Use the original link from your email
  • Check if you received a newer access email
  • Contact support if the problem persists
